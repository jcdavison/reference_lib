## Operators

### Arithmetic

```javascript
+
-
*
/
%
++ 
--
```

### Assignment

```javascript
= // x = y
+= // x += y, x = x + y
-= // x -= y, x = x - y
*= // x *= y, x = x * y
/= // x /= y, x = x / y
%= // x %= y, x = x % y 
```

### String concatenation

```javascript
"a" + "b" // "ab"
"5" + 5 // "55", NOTE the result is a string!
```


### Comparison

```javascript
== // equal
=== // equal value and equal type
!= // not equal
!== //not equal value or not equal type
> // greater than
< // less than
<=
>=
```

### Logic

```javascript
&& // and
|| // or
! // not
```

### `typeof`

```javascript
typeof 5 // number
typeof "5" // string
```
